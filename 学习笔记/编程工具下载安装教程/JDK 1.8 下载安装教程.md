# JDK 1.8 下载安装教程

## 1.下载

官网即可

```https
https://www.oracle.com/cn/java/technologies/javase-downloads.html
```

## 2.安装

## 3.配置环境变量

```
## JAVA_HOME  jdk安装目录
C:\Program Files\Java\jdk1.8.0_271

## path  bin目录
%JAVA_HOME%\bin\
```

## 4.查看是否配置完成

**注意：**记得先重启cmd

三个命令：

```
java -version
java
javac
```
